## Project setup

```bash
# config .env
$ change .env.example to .env

# install dependency
$ yarn install
```

## Compile and run the project

```bash
# run migration
$ yarn migration:run or yarn schema:sync

# watch mode
$ yarn run start:dev
```

## Open documentation 

```bash
# docs
$ open http://localhost:3001/api/docs
```
